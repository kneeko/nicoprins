Radio Hyrule

  • roleProgramming and design
  • toolsCordova, JavaScript, HTML/CSS

I built this app for Nyquist Studios. It's a simple radio player for their community web radio station, Radio Hyrule. Users can browse the entire song library to make requests as well as receive a notification when their song is up next.

Although this release only targeted Android, we opted to use Cordova in order to get a head start on the iOS and web redesign.
